Detailed Description

Central pinout and electrical configuration for the labyrinth table.

This board support package keeps ESP-IDF and board-specific constants out of the application layer. Change this file when the wiring changes.

esp_err_t bsp_board_init ( void )

Initialize basic board services.

The current first delivery does not require global board startup beyond component-specific initialization, but this hook gives the application a stable entry point for future board setup.

Returns
ESP_OK on success.
